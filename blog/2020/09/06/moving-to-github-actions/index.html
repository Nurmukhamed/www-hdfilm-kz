Сборка блога переехала с Travis CI на Github Actions

После выхода Github Actions возник - А зачем мне теперь Travis CI??? Задал себе вопрос и отложил его в ТОДО-списке.

И вот в воскресное утро решил, что пришло время.

  • Прочитал статью;
  • удалил файлы с секретами, которые нужно передавать в Travis CI;
  • добавил простой workflow для сборки блога на Github Actions;
  • cделал комиты, отправил (push) в Github;
  • все собралось.

Итог:

  • Удалились файлы с секретами;
  • Все работает в одной экосистеме;
  • Быстрее стала сборка.
